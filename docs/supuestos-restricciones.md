## 4. Registro de supuestos y restricciones

| ID | Categoría | Supuesto / Restricción | Estado | Comentarios |
|----|----------|----------------------|--------|-------------|
| S1 | Datos | Se asume que se dispone de datos simulados de cursos y horarios | Activo | No se cuenta con datos reales |
| S2 | Usuarios | Se asume que los estudiantes tienen acceso a internet | Activo | Necesario para el uso del sistema |
| S3 | Tecnología | Se asume uso de navegadores modernos | Activo | Compatible con Chrome y Edge |
| S4 | Uso | Se asume que los usuarios tienen conocimientos básicos de uso web | Activo | Facilita la interacción |
| R1 | Tiempo | Tiempo limitado para el desarrollo del proyecto | Activo | Condición académica |
| R2 | Datos | No se cuenta con acceso a datos reales de la universidad | Activo | Uso de datos simulados |
| R3 | Recursos | Recursos tecnológicos limitados | Activo | Desarrollo como prototipo |
| R4 | Alcance | El sistema será desarrollado como prototipo | Activo | No producción |
