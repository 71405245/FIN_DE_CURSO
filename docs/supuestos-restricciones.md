## 4. Registro de supuestos y restricciones

### 📌 Definición:
---

**Supuesto:** Algo que se considera cierto para los propósitos del proyecto.  

**Restricción:** Un factor limitante o restrictivo que afecta el desarrollo del sistema.  

**Registro de supuestos:** Documento dinámico que se utiliza para identificar y dar seguimiento a los supuestos y restricciones del proyecto.
