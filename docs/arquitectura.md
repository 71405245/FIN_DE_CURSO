# 🏗 Arquitectura del Sistema

## 🔹 Descripción

El sistema está basado en una arquitectura web cliente-servidor.

## 🔹 Tecnologías

- Lenguaje: Python  
- Framework: Django  
- Base de datos: MySQL / SQL Server  
- Frontend: HTML, CSS, Bootstrap 5  
- Generación de PDF: ReportLab / xhtml2pdf  

## 🔹 Componentes

- Backend: Lógica del sistema y procesamiento de datos  
- Frontend: Interfaz de usuario  
- Base de datos: Almacenamiento de información  
- Módulo de IA: Generación y recomendación de horarios  

## 🔹 Flujo

1. Usuario ingresa preferencias  
2. Sistema procesa datos  
3. Genera combinaciones  
4. Recomienda horarios  
5. Usuario visualiza o exporta
