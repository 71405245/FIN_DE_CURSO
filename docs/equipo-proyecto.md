## 3. Declaración del Proyecto

| Campo | Descripción |
|------|------------|
| **Nombre del Proyecto** | Sistema Web Inteligente para la Optimización del Proceso de Matrícula Universitaria |
| **Patrocinador del Proyecto** | Universidad Continental |
| **Responsable del Proyecto** | _ _ _ |
| **Fecha de Inicio** | 2026 |
| **Última Actualización** | 2026 |

---

| **Descripción del Proyecto** |
|-----------------------------|
| El proyecto consiste en el desarrollo de un sistema web inteligente que permite optimizar el proceso de matrícula universitaria mediante la generación y recomendación automática de horarios académicos, considerando las preferencias y restricciones del estudiante. |

---

| **Alcance del Proyecto** |
|--------------------------|
| El sistema permitirá generar horarios automáticamente, recomendar combinaciones óptimas, visualizar horarios y exportarlos en formato PDF. No incluye integración con sistemas reales ni matrícula oficial. |

---

| **Justificación del Proyecto** |
|-------------------------------|
| El proyecto busca mejorar la eficiencia del proceso de matrícula, reducir errores en la selección de cursos y optimizar el tiempo del estudiante mediante el uso de algoritmos de recomendación. |

---

| **Restricciones del Proyecto** | Detalle |
|-------------------------------|--------|
| **Tiempo** | Desarrollo limitado al periodo académico |
| **Recursos** | Recursos tecnológicos limitados |
| **Alcance** | Desarrollo como prototipo |
| **Datos** | Uso de datos simulados |

---

| **Entregables del Proyecto** |
|-----------------------------|
| Sistema web funcional |
| Módulo de recomendación de horarios |
| Generación y exportación de horarios |
| Documentación del sistema |

---

| **Beneficios del Proyecto** | Meta |
|---------------------------|------|
| Reducción del tiempo de matrícula | ≤ 3 minutos |
| Mejora en la toma de decisiones | Alta |
| Reducción de errores | Significativa |
